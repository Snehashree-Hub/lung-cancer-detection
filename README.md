# 🧠 Lung Cancer Detection using Deep Learning

## 📌 Description
This project detects lung cancer from CT scan images using a Convolutional Neural Network (CNN).

## 🛠️ Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy

## 🚀 Features
- Image classification (Cancer / No Cancer)
- Grad-CAM visualization
- Deep learning model (CNN)
